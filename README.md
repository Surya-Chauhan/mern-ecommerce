# 🛒 MERN E-Commerce Platform

A full-stack E-Commerce application built using the MERN stack (MongoDB, Express.js, React.js, and Node.js). The platform provides a complete online shopping experience with secure authentication, product management, shopping cart functionality, order processing, online payment integration, and an administrative dashboard for managing products and customer orders.

This project was developed to demonstrate practical experience in full-stack web development, RESTful API design, authentication and authorization, payment gateway integration, state management, database operations, and cloud deployment.

## 📖 Project Overview

The application enables customers to browse products, search and filter items, manage their shopping cart, place orders, and complete purchases through a secure payment gateway. Administrators can manage product inventory, update product information, monitor orders, and maintain featured products through a dedicated admin dashboard.

The project follows modern development practices, including component-based architecture, centralized state management, secure authentication, responsive design, and cloud deployment.

## 💳 Payment Integration

Integrated with Razorpay to support secure online transactions.

### Payment Flow

1. User proceeds to checkout.
2. Backend creates a Razorpay order.
3. User completes payment.
4. Payment signature is verified on the backend.
5. Order is successfully created and stored in the database.
